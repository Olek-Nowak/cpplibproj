Opis szczegółowy

template<class T>
class drzewo< T >

Implementacja binarnego drzewa poszukiwań.

Standardowa dynamiczna struktura danych, alokacja i dealokacja pamięci przebiegają automatycznie. Hierarchia elementów jest automatycznie zachowywana. Implementacja wyważania rozmiaru lub wysokości drzewa ułatwiona dzięki gotowym metodom @
